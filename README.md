# 💼 Job Portal Backend (MERN Stack)

A full-featured **Job Portal Backend API** built with Node.js, Express, and MongoDB.  
It supports authentication, job postings, applications, resume uploads, company management, ATS system, and job recommendations.

---

## 🚀 Features

### 🔐 Authentication & Authorization
- User registration (Job Seeker / Employer)
- Login / Logout
- JWT authentication
- Role-based access control (RBAC)
- Forgot / Reset password
- Email verification

---

### 👤 User Management
- Get profile
- Update profile
- Upload profile picture
- Soft delete account

---

### 🏢 Company Management
- Create / Update company
- Upload company logo
- Get company list
- Company profile page
- Admin / Owner control

---

### 💼 Job Management
- Create job postings
- Update / delete jobs
- Search & filter jobs
- Pagination & sorting
- Save / unsave jobs
- View job details

---

### 📄 Applications System
- Apply for jobs
- Upload resume
- Track application status
- Withdraw application
- Employer ATS dashboard
- Interview scheduling

---

### 📂 Resume System
- Upload resumes (PDF/DOC)
- Manage multiple resumes
- Set default resume
- Delete resumes

---

### 🤖 Recommendation System
- Job recommendations based on:
  - Skills
  - Location
  - Job type
  - Salary expectations
- Similar jobs feature

---

### 📧 Email System
- Password reset emails
- Application status notifications
- Email verification

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- Multer (file uploads)
- Cloudinary (optional)
- Nodemailer (email service)
